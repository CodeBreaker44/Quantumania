Used Algorithm

NTRUEncrypt: As quantum computers approach the regime where Shor's algorithm can efficientlys factor large integers and compute discrete logarithms, traditional cryptographic primitives such as RSA and ECC are rendered vulnerable. This imminent threat necessitates the exploration of post-quantum cryptographic solutions, and among the contenders, NTRUEncrypt emerges prominently. NTRUEncrypt is a lattice-based public key cryptosystem, underpinned by the hardness of the Ring Learning With Errors (Ring-LWE) problem. Unlike factorization and discrete logarithm problems, the Ring-LWE problem does not currently have an efficient quantum algorithmic solution. Within the context of FPGA F1 AWS instances, which offer vast parallelization and reconfigurability, NTRUEncrypt can be optimally implemented. This not only ensures high-throughput cryptographic operations but also provides a robust foundation for quantum-resistant security protocols in cloud-based infrastructures.
